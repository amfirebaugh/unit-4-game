Crystal Collector Game

At the start of the game, a randomly generated number will appear below. There are four crystals that represent a random number each round. Each time you click on a crystal you add points to your current score. The goal is to match your current score to the random number each round. If you can match the random number you win that round, and if you go over the random number then you lose that round. Each round the value of each crystal changes. Good luck!
